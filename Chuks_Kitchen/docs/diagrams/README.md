# Diagrams

This folder contains submission-ready backend diagrams for Chuks Kitchen.

## Files
- `user-registration-flow.md` - signup and OTP verification flow with edge cases.
- `order-processing-flow.md` - cart validation and order creation flow.
- `order-status-lifecycle.md` - strict order status state machine.
- `system-sequence.md` - API-level sequence of customer/admin/backend actions.
- `erd.md` - entity relationship diagram.
